# Ảnh sự kiện

Đặt ảnh sự kiện thật vào đúng thư mục này (`public/images/events/`) với **đúng tên file** dưới đây. Cho đến khi có ảnh thật, trang `/su-kien/` sẽ tự hiển thị ảnh minh họa (SVG trừu tượng theo màu thương hiệu) kèm chú thích tên file cần thay.

## Danh sách file cần thêm

| File cần đặt tên | Sự kiện |
|---|---|
| `workshop-sua-ao.jpg` | Workshop: Sửa áo cùng Ametia Green |
| `diem-thu-hoi-luu-dong-q7.jpg` | Điểm thu hồi lưu động - Quận 7 |
| `ngay-hoi-doi-do-cu.jpg` | Ngày hội đổi và thu hồi quần áo cũ |
| `buoi-tro-chuyen-hanh-trinh-mot-chiec-ao.jpg` | Buổi trò chuyện: Hành trình một chiếc áo |
| `workshop-cham-soc-vai-tai-che.jpg` | Workshop: Chăm sóc vải tái chế đúng cách |

## Yêu cầu kỹ thuật

- Tỷ lệ khung hình: ngang (wide), khoảng 3:2.
- Chiều rộng tối thiểu: 1200px.
- Định dạng: `.jpg`.

## Quan trọng

**Không** crop, resize, nén lại hay chỉnh sửa ảnh gốc bạn cung cấp. Chỉ cần đặt đúng tên file vào thư mục này.

Nếu bạn thêm sự kiện mới vào `src/data/events.ts`, nhớ cập nhật bảng trên với tên file `imageSlug` tương ứng.
