# Ảnh sản phẩm

Đặt ảnh sản phẩm thật vào đúng thư mục này (`public/images/products/`) với **đúng tên file** dưới đây. Cho đến khi có ảnh thật, trang web sẽ tự hiển thị ảnh minh họa (SVG trừu tượng theo màu thương hiệu) kèm chú thích tên file cần thay.

Không cần sửa code: chỉ cần thêm đúng tên file, ảnh sẽ tự động được dùng ở trang chủ và trang sản phẩm tương ứng.

## Danh sách file cần thêm

| File cần đặt tên | Sản phẩm |
|---|---|
| `ao-thun-form-rong.jpg` | Áo thun form rộng |
| `ao-so-mi-tay-ngan.jpg` | Áo sơ mi tay ngắn |
| `ao-khoac-mong.jpg` | Áo khoác mỏng |
| `quan-dai-ong-suong.jpg` | Quần dài ống suông |
| `quan-short.jpg` | Quần short |
| `tui-tote.jpg` | Túi tote |
| `mu-luoi-trai.jpg` | Mũ lưỡi trai |

Ngoài ra, trang chủ dùng thêm một ảnh hero chung: `hero-collection.jpg` (ảnh đại diện cho cả bộ sưu tập Green Capsule).

## Yêu cầu kỹ thuật

- Tỷ lệ khung hình: dọc (portrait), khoảng 4:5.
- Chiều rộng tối thiểu: 1200px.
- Định dạng: `.jpg` (có thể đổi sang `.jpg` nếu ảnh gốc là định dạng khác).

## Quan trọng

**Không** crop, resize, nén lại hay chỉnh sửa ảnh gốc bạn cung cấp. Chỉ cần đặt đúng tên file vào thư mục này, trang web sẽ tự hiển thị đúng kích thước qua CSS mà không làm biến dạng ảnh gốc.
